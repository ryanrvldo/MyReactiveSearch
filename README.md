# My Reactive Search

Coroutines Flow playground with [MapBox API](https://mapbox.com/)

## Configuration

1. Get your access token from [here](https://account.mapbox.com/auth/signup/), if you don't have a Mapbox account.
2. Add access token variable in your `local.properties` like
    ```
    ACCESS_TOKEN="YOUR_ACCESS_TOKEN"
    ```
3. Build and run your project. (make your access token variable is builded in `BuildConfig`)
